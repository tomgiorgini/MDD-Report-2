# MDD-Report-2
Here follows an explanation of the files and folders present in this repository:

- ARFFs Folder: the 4 ARFFs files

- Results Folder: all the results of the classification experiments, divided firstly in the 5 ensemble methods, then for each of these method divided in the 4 ARFFs. In each of the 20 folders there is a ranking.txt file.

Results.txt: all rankings on accuracies for each ensemble method and ARFF

Best_result.txt: the classification results for the 20 combinations of ARFFs + Ensemble Method, described in chapter 5 of the document
